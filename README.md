# 🍨 tools 🍨

一个用于 Windows 最佳包管理器 [Scoop](https://github.com/ScoopInstaller/Scoop)的自用脚本仓库

添加仓库：

```powershell
scoop bucket add tools https://github.com/wenkit1/tools
```
